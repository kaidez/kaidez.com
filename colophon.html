Colophon

kaidez.com was created by Kai Gittens using the following tools:

  • Jekyll
  • Sublime Text
  • Adobe Photoshop & Adobe Illustrator
  • 17" MacBook Pro
  • Grunt
  • iTerm

Posts on kaidez.com are written in Markdown within HTML5-based templates, with Liquid tags lending a helping hand. The site's source code is open source and freely available on GitHub.

On the development level, kaidez.com site dependencies are managed by Bower.

kaidez.com hosted on a LAMP stack provided by Media Temple. Static assets are hosted via MaxCDN.

kaidez.com doesn't care much about being viewed properly on Internet Explorer 8 and lower.
